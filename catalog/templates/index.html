Local Movies Library

Welcome to The Local Movies Library, a very basic Django website developed whit the help of this tutorial example on the Mozilla Developer Network, and OpenBootcamp.

The tutorial demonstrates how to create a Django skeleton website and application, define URL mappings, views (including Generic List and Detail Views), models and templates.

This site is more basic than the one developed in the tutorial, and it's related to movies as it's the resolution of the 12th chapter's exercice of python's course in OpenBootcamp.
